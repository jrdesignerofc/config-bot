# Jr Store Config Bot

Versão ajustada para Railway.

## Estrutura esperada no repositório
Os arquivos `bot.py`, `main.py`, `requirements.txt` e a pasta `data/` devem ficar na raiz do projeto.

## Railway
Use um destes métodos:

1. Deixe o Railway detectar o `Procfile` automaticamente.
2. Ou configure manualmente o Start Command como:
   `python bot.py`

## Variáveis
- DISCORD_TOKEN
- CONFIG_CHANNEL_ID

Opcional:
- GUILD_ID
- BRAND_NAME
- BRAND_COLOR
